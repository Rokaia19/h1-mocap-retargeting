# Scripts

This folder will contain the project code.

Planned scripts:
- `load_mocap.py`: load and inspect motion capture data
- `retarget_kinematics.py`: retarget human motion to H1 kinematics
- `visualize_motion.py`: visualize human and retargeted robot motion
- `stability_analysis.py`: analyze CoM, support polygon, and ZMP
- `run_mujoco_sim.py`: simulate the motion in MuJoCo
- `optimal_control.py`: improve motion smoothness and stabilization
